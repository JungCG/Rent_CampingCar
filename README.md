# Rent_CampingCar
- Manage and Rent CampingCar
- JDBC, Swing을 이용한 CampingCar Office 구현 (Customer / Manager)


## Using
**1. DBMS** : MySQL 8.0.19

**2. 언어** : Java(JDK 1.8)

**3. IDE** : Eclipse
